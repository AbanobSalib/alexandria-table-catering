# Alexandria Table

Static catering website for a home-based Egyptian chef in Alexandria, Egypt.

## Launch checklist

1. Add the chef's WhatsApp number in `script.js` using international format without `+`, for example `2010...`.
2. Replace the placeholder brand name if your friend wants to use her real name.
3. Add real food photos when available. The current images are remote placeholder food photography.
4. Review menu prices before taking orders because ingredient prices change.
5. Create the social profiles from `social-launch-kit.md`, then update the footer links.

## GitHub Pages

This site is built as plain HTML, CSS, and JavaScript. It can be hosted directly with GitHub Pages from the `main` branch root.
